## Prerequisites

Before you begin, ensure you have **GetSimple CMS** installed on your server. If not, you can download it from the [official website](http://get-simple.info/).

---

## Installation

1. **Download the Theme:**
   - Clone or download this repository as a `.zip` file.

2. **Move to Theme Folder:**
   - Extract the downloaded theme and place the folder inside the `themes` directory of your GetSimple CMS installation. 
     ```
     /path-to-getsimple-cms/themes/apri-crm
     ```

3. **Activate the Theme:**
   - Log in to your GetSimple CMS admin panel.
   - Navigate to **Theme > Themes**.
   - Select `apri-crm` from the available options and click "Activate."

---

## Usage Guidelines

Follow the standard **installation and usage policies of GetSimple CMS** for theme management and customization. You can refer to the [GetSimple Documentation](http://get-simple.info/docs/) for more information.
